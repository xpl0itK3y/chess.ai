# Chess.ai - Шахматное приложение на React

Интерактивное шахматное приложение, написанное на React с использованием TypeScript. Приложение включает в себя полную игровую логику, таймер и отслеживание сбитых фигур.

## Требования

- **Node.js** версии 14.x или выше
- **npm** версии 6.x или выше
- **Docker** и **Docker Compose** (для запуска через Docker)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd chess.ai
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск приложения

```bash
npm start
```

Приложение будет доступно по адресу: `http://localhost:3000`

### 4. Другие полезные команды

```bash
# Сборка приложения для продакшена
npm run build

# Запуск тестов
npm test

# Выход из create-react-app (необратимо)
npm run eject
```

## Запуск через Docker

### Вариант 1: Использование Docker Compose (рекомендуется)

```bash
# Запуск приложения в фоновом режиме
docker-compose up -d

# Запуск с отображением логов
docker-compose up

# Остановка приложения
docker-compose down
```

### Вариант 2: Использование Docker

```bash
# Сборка образа
docker build -t chess-app .

# Запуск контейнера
docker run -p 3000:3000 chess-app
```

Приложение будет доступно по адресу: `http://localhost:3000`

## Особенности

- **Полная шахматная логика**: Реализованы все правила ходов для каждой фигуры
- **Интерфейс на React/TypeScript**: Современный и типизированный код
- **Таймер игры**: Отслеживание времени для каждого игрока
- **Отображение сбитых фигур**: Визуализация съеденных фигур
- **Горячая перезагрузка**: При запуске через Docker Compose
- **Адаптивный дизайн**: Корректное отображение на разных устройствах

## Важные замечания

- При запуске через Docker Compose исходный код монтируется для поддержки горячей перезагрузки
- Порт по умолчанию: 3000

## Как играть

1. Белые фигуры всегда начинают игру
2. Кликните на фигуру, чтобы выбрать её
3. Кликните на целевую клетку, чтобы сделать ход
4. Таймер отслеживает время каждого игрока
5. Сброшенные фигуры отображаются сбоку от доски
